### Описание проекта

Проведение исследования рынка заведений общественного питания Москвы для выбора подходящего инвесторам места открытия нового заведения (кофейни)

### Навыки и инструменты

- **python**
- **pandas**
- **matplotlib**
- **numpy**
- **seaborn**
- **plotly**
- **PowerPoint**

### Вывод

Проведен анализ данных по кофейням Москвы в разных разрезах, предложены рекомендации по району открытия, часам работы, количсетву посадочных мест и ценой за чашку кофе.

Подготовлен презентационный материал по результатам исследования.
